# Bike Sales – Excel Dashboard

This project analyses Bike Sales performance across different European countries
using Microsoft Excel. The dashboard highlights Revenue %, Cost %, Profit % and
Profit Margin by country.

## Tools Used
• Microsoft Excel  
• Pivot Tables  
• Pivot Charts  
• Slicers (Year & Customer Age)  
• Calculated Fields  

## Key Insights
• Australia generates the highest share of Revenue (38%)  
• Germany and Canada have the highest Profit Margins  
• Overall business profit margin is ~40%
